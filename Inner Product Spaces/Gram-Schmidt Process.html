Gram-Schmidt Process

6.4. Gram-Schmidt Process#

We appreciate all the benefits and good properties orthonormal bases can bring us. But how can we construct such an orthonormal basis? Suppose we have a list of linearly independent vectors, then we can apply the Gram-Schmidt process, which will be introduced in the next theorem, to construct an orthonormal list of vectors out it with the same span.

Theorem 6.5 (Gram-Schmidt)

Let \(V\) be a finite-dimensional inner product space over \(\FF\). Suppose \((\mathbf{v}_1, \ldots, \mathbf{v}_m)\) is a list of linearly independent vectors in \(V\), then there exists a list of orthonormal vectors \((\mathbf{e}_1, \ldots, \mathbf{e}_m)\) in \(V\) such that

(6.13)#\[\begin{align}\Span(\mathbf{e}_1, \ldots, \mathbf{e}_j) = \Span(\mathbf{v}_1, \ldots, \mathbf{v}_j) \quad\forall 1 \leq j \leq m \end{align}\]

Proof. We shall prove by induction on \(m\).

(Base Case) If \(m = 1\), then the theorem holds trivially by letting \(\mathbf{e}_1 = \frac{1}{\norm{\mathbf{v}_1}} \mathbf{v}_1\).

(Inductive Step) Assume that this theorem holds for \(m = k\), we shall show that it also holds for \(m = k+1\). Let \((\mathbf{v}_1, \ldots, \mathbf{v}_{k+1})\) be a list of \(k+1\) linearly independent vectors in \(V\). Applying the induction hypothesis to the first \(k\) vectors, it follows that there exists as a list of orthonormal vectors \((\mathbf{e}_1, \ldots, \mathbf{e}_k)\) such that

(6.14)#\[\begin{align}\Span(\mathbf{e}_1, \ldots, \mathbf{e}_j) = \Span(\mathbf{v}_1, \ldots, \mathbf{v}_j) \quad\forall 1 \leq j \leq k \end{align}\]

We now construct the vector \(\mathbf{e}_{k+1}\). Define

(6.15)#\[\begin{align}\mathbf{u} = \mathbf{v}_{k+1} - \sum_{j=1}^k \left\langle\mathbf{e}_j, \mathbf{v}_{k+1}\right\rangle\mathbf{e}_j \end{align}\]

Note that \(\mathbf{u} \neq \mathbf{0}\). Assume \(\mathbf{u} = \mathbf{0}\), then (6.15) would imply that

\[\begin{align*}\mathbf{v}_{k+1}\in\Span(\mathbf{e}_1, \ldots, \mathbf{e}_k) = \Span(\mathbf{v}_1, \ldots, \mathbf{v}_k) \end{align*}\]

which contradicts that \((\mathbf{v}_1, \ldots, \mathbf{v}_{k+1})\) are linearly independent. Now, it is safe to define

(6.16)#\[\begin{align}\mathbf{e}_{k+1} = \frac{1}{\norm{\mathbf{u}}}\mathbf{u}\end{align}\]

It is clear \(\norm{\mathbf{e}_{k+1}} = 1\). We then need to verify that \(\mathbf{e}_{k+1}\) is orthogonal to each \(\mathbf{e}_j\)(\(1 \leq j \leq k\)). It follows from (6.15) and (6.16) that

\[\begin{align*}\left\langle\mathbf{e}_j, \mathbf{e}_{k+1}\right\rangle = \frac{1}{\norm{\mathbf{u}}}\left(\left\langle\mathbf{e}_j, \mathbf{v}_{k+1}\right\rangle - \left\langle\mathbf{e}_j, \mathbf{v}_{k+1}\right\rangle\cdot 1 \right)= 0 \quad\forall 1 \leq j \leq k \end{align*}\]

Therefore, \((\mathbf{e}_1, \ldots, \mathbf{e}_{k+1})\) is indeed a list of orthonormal vectors.

We also need to show (6.13) is satisfied for \(m = k+1\). Note that (6.15) and (6.16) imply that \(\mathbf{v}_{k+1} \in \Span(\mathbf{e}_1, \ldots, \mathbf{e}_{k+1})\). On the other hand, \(\mathbf{e}_{k+1}\) can be written as a linear combination of \(\mathbf{e}_1, \ldots, \mathbf{e}_k\) and \(\mathbf{v}_{k+1}\). Recall (6.14), and we will note that \(\mathbf{e}_{k+1}\) can also be written as a linear combination of \((\mathbf{v}_1, \ldots, \mathbf{v}_{k+1})\). Therefore,

\[\begin{align*}\mathbf{v}_{k+1}&\in\Span(\mathbf{e}_1, \ldots, \mathbf{e}_{k+1}) & \mathbf{e}_{k+1}&\in\Span(\mathbf{v}_1, \ldots, \mathbf{v}_{k+1}) \end{align*}\]

Combined with (6.14), we obtain that

\[\begin{align*}\Span(\mathbf{e}_1, \ldots, \mathbf{e}_j) = \Span(\mathbf{v}_1, \ldots, \mathbf{v}_j) \quad\forall 1 \leq j \leq k+1 \end{align*}\]

This completes the proof.

As proposed in the proof of Theorem 6.5, given a list of linearly independent vectors,

\[\begin{align*}\mathbf{v}_1, \ldots, \mathbf{v}_m \end{align*}\]

a list of orthonormal vectors can be found as follows.

(6.17)#\[\begin{split}\begin{align}\begin{array}{clccl} \mathbf{u}_1 &= \mathbf{v}_1 &\quad &\mathbf{e}_1 &= \frac{1}{\norm{\mathbf{u}_1}} \mathbf{u}_1 \\ \mathbf{u}_2 &= \mathbf{v}_2 - \proj_{\mathbf{e}_1} \mathbf{v}_2 = \mathbf{v}_2 - \left\langle\mathbf{e}_1, \mathbf{v}_2 \right \rangle \mathbf{e}_1 &\quad &\mathbf{e}_2 &= \frac{1}{\norm{\mathbf{u}_2}} \mathbf{u}_2 \\ \vdots&&&\vdots\\ \mathbf{u}_m &= \mathbf{v}_m - \sum_{j=1}^{m-1} \proj_{\mathbf{e}_j} \mathbf{v}_m = \mathbf{v}_m - \sum_{j=1}^{m-1} \left\langle\mathbf{e}_j, \mathbf{v}_m \right \rangle \mathbf{e}_j &\quad &\mathbf{e}_m &= \frac{1}{\norm{\mathbf{u}_m}} \mathbf{u}_m \end{array}\end{align}\end{split}\]

If \((\mathbf{v}_1, \ldots, \mathbf{v}_n)\) forms a basis for \(V\), then the list of vectors \((\mathbf{e}_1, \ldots, \mathbf{e}_n)\) given by (6.17) is an orthonormal basis for \(V\).

Given any basis of a finite-dimensional inner product space, we can always apply the Gram-Schmidt procedure to convert this basis to an orthonormal one.

Corollary 6.2

Any finite-dimensional inner product space has an orthonormal basis.

Proof. Let \(V\) be an \(n\)-dimensional inner product space, and \((\mathbf{v}_1, \ldots, \mathbf{v}_n)\) its basis. Theorem 6.5 says there exists a list of orthonormal vectors \((\mathbf{e}_1, \ldots, \mathbf{e}_n)\) such that

\[\begin{align*}\Span(\mathbf{e}_1, \ldots, \mathbf{e}_n) = \Span(\mathbf{v}_1, \ldots, \mathbf{v}_n) = V \end{align*}\]

And we know from Corollary 6.1 that \((\mathbf{e}_1, \ldots, \mathbf{e}_n)\) is linearly independent. (Or we can conclude it is linearly independent just based on the fact that it spans \(V\) with precisely \(n\) vectors.) Hence, it indeed forms a basis.

Recall Theorem 6.2. We know that we can extend a list of linearly independent vectors to a basis. Now, if we have a list of orthonormal vectors at hand, can we extend it to an orthonormal basis? Fortunately, we can do so by performing the Gram-Schmidt process.

Corollary 6.3

Every orthonormal list of vectors \((\mathbf{e}_1, \ldots, \mathbf{e}_m)\) in a finite-dimensional inner product space can be extended to an orthonormal basis.

Proof. If \(\dim V = m\), then we are already done. Suppose the dimension of \(V\) is larger than \(m\), say \(m + n\). By Theorem 6.2, there exists \(\mathbf{v}_1, \ldots, \mathbf{v}_n \in V\) such that

\[\begin{align*}(\mathbf{e}_1, \ldots, \mathbf{e}_n, \mathbf{v}_1, \ldots, \mathbf{v}_n) \end{align*}\]

forms a basis of \(V\). Then by applying the Gram-Schmidt procedure (Theorem 6.5) to this basis, we obtain an orthogonal list of vectors

\[\begin{align*}(\mathbf{e}_1^\prime, \ldots, \mathbf{e}_n^\prime, \mathbf{f}_1, \ldots, \mathbf{f}_n) \end{align*}\]

Since this list of vectors is linearly independent by Corollary 6.1, and it contains \(m+n = \dim V\) vectors, it must be a basis for \(V\).

What is left to show is that the first \(n\) vectors \((\mathbf{e}_1^\prime, \ldots, \mathbf{e}_n^\prime)\) remain unchanged. Referring to (6.17), we have

\[\begin{align*}\mathbf{e}_1^\prime = \mathbf{e}_1 \end{align*}\]

Having

\[\begin{align*}\mathbf{e}_i^\prime = \mathbf{e}_i \quad\forall i=1, \ldots,j-1 \end{align*}\]

where \(j \geq 2\), we can compute using (6.17) that

\[\begin{align*}\mathbf{e}_j^\prime = \mathbf{e}_j - \sum_{i=1}^{j-1}\left\langle\mathbf{e}_i^\prime, \mathbf{e}_j \right\rangle\mathbf{e}_i^\prime = \mathbf{e}_j - \sum_{i=1}^{j-1}\left\langle\mathbf{e}_i, \mathbf{e}_j \right\rangle\mathbf{e}_i = \mathbf{e}_j \end{align*}\]

Therefore, we have

\[\begin{align*}\mathbf{e}_j^\prime = \mathbf{e}_j \quad\forall j = 1, \ldots, m \end{align*}\]

This completes the proof.
