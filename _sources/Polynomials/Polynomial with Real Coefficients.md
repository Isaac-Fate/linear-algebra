# Polynomial with Real Coefficients

````{prf:theorem}
:label: thm:5

Let $p(x)$ be a nonconstant polynomial with real coefficients. Then it has a unique factorization (except for the order of factors) of the form

```{math}
\begin{align*}
p(x) = c (x - \lambda_1) \cdots(x - \lambda_k)
(x^2 + \alpha_1 x + \beta_1) \cdots(x^2 + \alpha_l x + \beta_l)
\end{align*}
```

where $k + m \geq 1$ and $\lambda_j, \alpha_j, \beta_j \in \R$ with $\alpha_j^2 < 4\beta_j$.
````
