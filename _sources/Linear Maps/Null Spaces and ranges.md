````{prf:theorem}
:label: thm:4

Let $T: V \to W$ be a linear map. Then $T$ is injective if and only if $\ker T = \{\mathbf{0}\}$.
````

````{prf:corollary}
:label: cor:1

Let $T: V \to W$ be a linear map. Then $T$ is injective if and only if $(T\mathbf{v}_1, \ldots, T\mathbf{v}_n)$ is linearly independent in $W$ as long as $(\mathbf{v}_1, \ldots, \mathbf{v}_n)$ is linearly independent in $V$. Mathematically,

```{math}
\begin{align*}(\mathbf{v}_1, \ldots, \mathbf{v}_n) \;\text{is linearly independent in $V$}\implies(T\mathbf{v}_1, \ldots, T\mathbf{v}_n) \;\text{is linearly independent in $W$}\end{align*}
```
````
